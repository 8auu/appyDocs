# Premium

| Command name     | Description                                                                                  |
| ---------------- | -------------------------------------------------------------------------------------------- |
| /premium\_add    | Allows [Patreon](https://www.patreon.com/appybot) subscribers to redeem their premium perks. |
| /premium\_remove | Allows patreon subscribers to unredeem their premium perks.                                  |
